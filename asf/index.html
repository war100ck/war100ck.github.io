Commands Access Description

 

 Master Generates temporary 2FA token for curren bot instances.

 

 Master Generates temporary 2FA token for given bot instances.

 

 Master Denies all pending 2FA confirmations for current bot instance.

 

 Master Denies all pending 2FA confirmations for given bot instances.

 

 Master Accepts all pending 2FA confirmations for current bot instance.

 

 Master Accepts all pending 2FA confirmations for given bot instances.

 

 Operator Activates given appIDs (Steam Network) or subIDs (Steam Store) on current bot instance (free games only).

 

 Operator Activates given appIDs (Steam Network) or subIDs (Steam Store) on given bot instances (free games only).

 

 Master Lists blacklisted users from trading module of current bot instance.

 

 Master Lists blacklisted users from trading module of given bot instances.

 

 Master Blacklists given steamIDs from trading module of current bot instance.

 

 Master Blacklists given steamIDs from trading module of given bot instances.

 

 Master Removes blacklist of given steamIDs from trading module of current bot instance.

 

 Master Removes blacklist of given steamIDs from trading module of given bot instances.

 

 Owner Stops whole ASF process.

 

 Master Restarts cards farming module for current bot instance.

 

 Master Restarts cards farming module for given bot instances.

 

 FamilySharing Shows help (link to this page).

 

 Master Sets given input type to given value for current bot instance, works only in Headless mode - further explained below.

 

 Master Sets given input type to given value for given bot instances, works only in Headless mode - further explained below.

 

 Master Lists apps blacklisted from automatic idling of current bot instance.

 

 Master Lists apps blacklisted from automatic idling of given bot instances.

 

 Master Adds given appIDs to apps blacklisted from automatic idling of current bot instance.

 

 Master Adds given appIDs to apps blacklisted from automatic idling of given bot instances.

 

 Master Removes given appIDs from apps blacklisted from automatic idling of current bot instance.

 

 Master Removes given appIDs from apps blacklisted from automatic idling of given bot instances.

 

 Master Lists priority idling queue of current bot instance.

 

 Master Lists priority idling queue of given bot instances.

 

 Master Adds given appIDs to priority idling queue of current bot instance.

 

 Master Adds given appIDs to priority idling queue of given bot instances.

 

 Master Removes given appIDs from priority idling queue of current bot instance.

 

 Master Removes given appIDs from priority idling queue of given bot instances.

 

 Master Makes current bot instance leave the group chat. For obvious reasons, this command works only in group chats.

 

 Master Makes given bot instances leave the group chat. For obvious reasons, this command works only in group chats.

 

 Master Sends all MatchableTypes items of current bot instance to Master user defined in its SteamUserPermissions (with lowest steamID if more than one).

 

 Master Sends all MatchableTypes items of given bot instances to Master user defined in their SteamUserPermissions (with lowest steamID if more than one).

 

 Master Changes Steam nickname of current bot instance to given nickname.

 

 Master Changes Steam nickname of given bot instances to given nickname.

 

 Operator Checks if current bot instance already owns given appIDs and/or gameNames (can be part of the game's name). It can also be * to show all games available.

 

 Operator Checks if given bot instances already own given appIDs and/or gameNames (can be part of the game's name). It can also be * to show all games available.

 

 Master Prints encrypted password of current bot instance (in use with PasswordFormat).

 

 Master Prints encrypted password of given bot instances (in use with PasswordFormat).

 

 Operator Permanently pauses automatic cards farming module of current bot instance. ASF will not attempt to farm current account in this session, unless you manually !resume it, or restart the process. Also called sticky pause.

 

 Operator Permanently pauses automatic cards farming module of given bot instances. ASF will not attempt to farm current account in this session, unless you manually !resume it, or restart the process. Also called sticky pause.

 

 FamilySharing Temporarily pauses automatic cards farming module of current bot instance. Farming will be automatically resumed on the next playing event, or bot disconnect. You can !resume farming to unpause it.

 

 FamilySharing Temporarily pauses automatic cards farming module of given bot instances. Farming will be automatically resumed on the next playing event, or bot disconnect. You can !resume farming to unpause it.

 

 Operator Temporarily pauses automatic cards farming module of current bot instance for given amount of seconds. After delay, cards farming module is automatically resumed.

 

 Operator Temporarily pauses automatic cards farming module of given bot instances for given amount of seconds. After delay, cards farming module is automatically resumed.

 

 Master Switches to manual farming - launches given appIDs on current bot instance. Use !resume for returning to automatic farming.

 

 Master Switches to manual farming - launches given appIDs on given bot instances. Use !resume for returning to automatic farming.

 

 Operator Redeems given cd-keys on current bot instance. !redeem XXXXX-XXXXX-XXXXX,XXXXX-XXXXX-XXXXX,XXXXX-XXXXX-XXXXX,XXXXX-XXXXX-XXXXX,XXXXX-XXXXX-XXXXX,...

 

 Operator Redeems given cd-keys on given bot instances.

 

 Operator Redeems given cd-keys on current bot instance, using given modes explained below.

 

 Operator Redeems given cd-keys on given bot instances, using given modes explained below.

 

 Operator Forces current bot instance to rejoin its SteamMasterClanID group chat.

 

 Operator Forces given bot instances to rejoin their SteamMasterClanID group chat.

 

 Owner Restarts ASF process.

 

 FamilySharing Resumes automatic farming of current bot instance. Also see !pause, !play.

 

 FamilySharing Resumes automatic farming of given bot instances. Also see !pause, !play.

 

 Master Starts given bot instances.

 

 Owner Prints process statistics, such as managed memory usage.

 

 FamilySharing Prints status of current bot instance.

 

 FamilySharing Prints status of given bot instances.

 

 Master Stops current bot instance.

 

 Master Stops given bot instances.

 

 Master Sends from current bot instance to given botinstance, all inventory items of that are matching given modes, explained below.

 

 Master Sends from given bot instances to given botinstance, all inventory items of that are matching given modes, explained below.

 

 Master Unpacks all booster packs stored in the inventory of current bot instance.

 

 Master Unpacks all booster packs stored in the inventory of given bot instances.

 

 Owner Checks GitHub for ASF updates (this is done automatically every 24 hours if AutoUpdates).

 

 FamilySharing Prints version of ASF.
